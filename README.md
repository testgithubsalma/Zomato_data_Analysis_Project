# Zomato_data_Analysis_Project

# 🍽️ Zomato Data Analysis 📊 | SQL, Power BI

An interactive data analysis project that explores Zomato restaurant data to uncover insights about ratings, pricing, locations, and customer preferences. This project focuses on transforming raw data into meaningful visual insights using SQL and Power BI.

# 🔍 Overview

Food delivery platforms generate large amounts of data. This project analyzes Zomato dataset to understand:

Restaurant performance based on ratings
Cost distribution across different locations
Popular cuisines and customer preferences
Location-wise trends and demand patterns

The goal is to convert raw data into actionable insights through data cleaning, querying, and visualization.

# 🚀 Features

# 📊 Interactive Power BI Dashboard
📍 Location-based analysis of restaurants
⭐ Rating distribution and insights
💰 Cost analysis and pricing trends
🍴 Cuisine-based insights
🧹 Data cleaning and transformation using SQL
📈 Clear and visually appealing reports
